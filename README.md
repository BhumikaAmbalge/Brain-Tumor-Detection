# 🧠 Brain Tumor Detection using Deep Learning

This project focuses on detecting brain tumors from MRI scans using a Convolutional Neural Network (CNN) model.  
A MERN-based web application is developed to allow users to upload MRI images and receive an instant prediction about whether a tumor is present.

---

## 🚀 Features
- Upload MRI images to detect tumor presence
- Deep Learning model integrated with backend
- Real-time prediction
- Modern and responsive UI
- Secure and fast handling of medical images

## 🛠️ How It Works
1. MRI image is uploaded through the web interface
2. Image is preprocessed and passed to trained CNN model
3. Model outputs prediction as **Tumor** or **No Tumor**
4. Result displayed on UI instantly
